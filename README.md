# HTML Slide Presentation

This project contains HTML-based presentation slides designed for the Wrtn AX Team Intern Assessment.

## Project Structure

- `슬라이드/`: Contains the individual HTML slide files.
- `navigation.js`: Handles keyboard navigation (Arrow keys) between slides.
- `index.html`: Entry point for GitHub Pages.

## How to View

### Online (GitHub Pages)
Visit the GitHub Pages URL (once configured) and append `/슬라이드/` to viewing the slides.

### Local
1. Open `슬라이드/index.html` or any `슬라이드X.html` file in your browser.
2. Use **Left/Right Arrow keys** to navigate.
