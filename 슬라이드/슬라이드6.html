성공 실험 ① Hybrid Search

검색 전략의 시행착오 끝에 찾아낸 최적의 조합 (From Vector to WRRF)

Phase 01

Vector Only

의미 기반 검색만 사용.
법률 용어의 정확한 매칭보다
맥락적 유사도에 의존.

Accuracy 51.35% Baseline Level
Phase 02

Hybrid Search 적용

단순 RRF 및 가중합(Weighted) 시도.
오히려 검색 노이즈가 증가하며
성능이 정체되거나 하락함.

Accuracy 50.19% Performance Drop
Phase 03 (Final)

WRRF 적용 및 최적화

Weighted Reciprocal Rank Fusion 적용.
순위(Rank)와 가중치(0.7/0.3)를
동시에 고려하여 최적화.

Accuracy 56.76% +5.18%p Boost

CORE LESSON

단순 결합이 아닌, 순위(Rank)의 안정성과 가중치(Weight)의 유연성을 결합했을 때 시너지가 발생했습니다.
